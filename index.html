立碼驗服務(eDDA)申請流程

步驟1：取得專屬QR Code

點選「線上申請」→ 輸入基本資料與約定扣繳銀行帳號 → 下載QR Code或拍照留存。

步驟2：二天內至便利超商機台執行「立碼驗」插卡驗證

攜帶個人專屬「QR Code」與約定扣繳銀行「晶片金融卡」至超商機台 (全家、萊爾富、7-ELEVEN) 進行插卡驗證。

  • 全家
  • 萊爾富
  • 7-ELEVEN
全家便利商店_FamiPort 立碼驗流程

1點選「會員」服務

2選擇「立碼驗」服務功能

3閱讀超商服務條款

4掃描玉山銀行「專屬QR Code」

5插入約定扣繳銀行「晶片金融卡」

6輸入卡片密碼，正確即可完成申辦
